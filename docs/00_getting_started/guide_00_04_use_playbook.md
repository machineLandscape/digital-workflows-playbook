# GUIDE_00_04 — How to Use This Playbook in Studio

**Version:** v01.0  
**Audience:** Students and instructors  
**Last updated:** 2025-12-20  

---

## 1. What this playbook is

This is **not a course** and **not a linear tutorial**.

It is a **reference system** for digital work in studio.

---

## 2. How to use it

- Start where you are stuck
- Use guides independently
- Do not read everything

This mirrors professional practice.

---

## 3. Common entry points

- No data → GIS_01
- Messy model → RHINO_01
- Broken drawings → CAD_03 / CAD_04
- Ugly graphics → ADOBE guides

---

## 4. What to ignore

Ignore:
- Tools you are not using
- Guides unrelated to your task
- Perfection

Focus on progress.

---

## 5. For instructors

- Assign guides as references, not homework
- Use class time for critique and discussion
- Let students self-diagnose

---

## 6. Philosophy

Digital workflow is **infrastructure**.

Once it works, design gets easier.
