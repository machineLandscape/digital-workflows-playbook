# GUIDE_00_03 — Troubleshoot When Something Goes Wrong

**Version:** v01.0  
**Audience:** Mixed ability  
**Last updated:** 2025-12-20  
**Applies to:** All software and all studios  
**Does not cover:** Software-specific commands

---

## 1. What this does

This guide helps you **diagnose and recover from common digital workflow problems** before they become crises.

---

## 2. When to use this

Use this guide when:
- A file will not open
- Something disappeared
- An export looks wrong
- You feel stuck and frustrated

---

## 3. Stop first

Before doing anything else:
- Save your file
- Stop clicking randomly
- Do not start over yet

Most problems are recoverable.

---

## 4. Identify the problem type

Which best fits?

- **File problem** (missing, overwritten, broken links)
- **Display problem** (invisible objects, wrong scale)
- **Translation problem** (things break between software)
- **Output problem** (PDF/image wrong)

---

## 5. Basic recovery steps

1. Check folder structure  
2. Open the most recent *working* version  
3. Zoom to extents  
4. Turn layers on/off  
5. Reduce complexity  

---

## 6. Use the right fix

| Problem | Go to |
|---|---|
| File chaos | GUIDE_00_01 |
| GIS not aligning | GIS → CAD |
| Model export wrong | Rhino → CAD |
| PDF broken | CAD export |

---

## 7. Checkpoint

You should be able to say:
- What kind of problem this is
- What changed last
- Which guide addresses it
